# 🔧 Sistema Cadastro Java - CRUD Completo

## 🛠️ Tecnologias
Java 17 | MySQL | POO | JDBC | Swing

## ✨ Funcionalidades
- Cadastro de clientes
- Listagem de dados
- Edição de informações
- Exclusão de registros
- Validação de campos

## 📁 Estrutura
src/
 ├── model
 ├── dao
 ├── view
 └── main

## 🚀 Como executar
1. Criar banco no MySQL:
CREATE DATABASE cadastro_java;

2. Rodar o projeto:
Main.java
